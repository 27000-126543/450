## 1. 产品概述

全国性通信光缆网络运维与故障预警分析平台，面向电信运营商集团、省公司及地市运维团队，提供光缆健康实时监测、智能预警分级处置、巡检路线优化和运维诊断报告自动化生成的全链路解决方案，旨在降低光缆故障平均修复时长、提升纤芯利用率、实现从被动抢修到主动预防的运维模式转型。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 层级 | 核心权限 |
|------|------|----------|
| 集团管理员 | 集团级 | 查看全国所有光缆明细与统计报表、管理省级账号、审批跨省割接方案 |
| 省公司网络部 | 省级 | 查看所辖省份光缆明细与报表、批准属地割接方案、管理市级账号 |
| 区域经理 | 省级（下属） | 复核二级预警、审核割接方案、调度巡检资源 |
| 运维班组长 | 市级 | 接收一级预警推送、确认/处置预警、上传巡检数据、查看所辖地市光缆明细 |
| 运维工程师 | 市级（下属） | 执行巡检任务、记录故障处置、查看分配任务 |

### 2.2 功能模块

1. **全国总览看板**：全国光缆健康热力图、故障排名、按省份/运营商切换、核心指标卡片（健康指数、故障率、平均修复时长）
2. **省份下钻页**：地市光缆明细、近30天光功率趋势曲线、故障类型分布饼图、光缆段列表筛选
3. **预警中心**：实时预警列表、预警分级（一级/二级）、三级审批流程（班组长确认→区域经理复核→省公司网络部批准）、预警处置时间线
4. **巡检管理**：年度巡检计划Excel上传、关键段落自动提取、未来90天高风险光缆段预测、最优巡检路线推荐、备纤熔接方案推荐
5. **运维报告**：每周自动生成运维诊断报告、故障率同比环比、修复时长排名、纤芯利用率、趋势对比与策略推荐

## 3. 核心流程

### 3.1 预警生成与处置流程

光缆段数据实时监测 → 光功率连续下降超阈值或振动异常 → 系统自动生成一级预警 → 推送至属地运维班组长 → 班组长2小时内确认处置 →（若未处置）自动升级为二级预警 → 启动三级审批流程 → 班组长确认 → 区域经理复核 → 省公司网络部批准 → 执行割接方案调整

```mermaid
flowchart TD
    "实时数据监测" --> "异常检测"
    "异常检测" --> "生成一级预警"
    "生成一级预警" --> "推送班组长"
    "推送班组长" --> "2小时内处置?"
    "2小时内处置?" -->|"是"| "预警关闭"
    "2小时内处置?" -->|"否"| "升级二级预警"
    "升级二级预警" --> "班组长确认"
    "班组长确认" --> "区域经理复核"
    "区域经理复核" --> "省公司批准"
    "省公司批准" --> "执行割接方案"
    "执行割接方案" --> "预警关闭"
```

### 3.2 巡检计划与预测流程

上传年度巡检计划Excel → 系统自动提取关键段落 → 结合历史数据预测未来90天高风险段 → 推荐最优巡检路线 → 推荐备纤熔接方案 → 运维班组长确认执行

```mermaid

    "上传巡检计划Excel" --> "自动提取关键段落"
    "自动提取关键段落" --> "结合历史数据预测"
    "结合历史数据预测" --> "生成高风险段列表"
    "生成高风险段列表" --> "推荐最优巡检路线"
    "推荐最优巡检路线" --> "推荐备纤熔接方案"
    "推荐备纤熔接方案" --> "班组长确认执行"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝（#0A1628）+ 青蓝色（#00D4FF）科技感主题，搭配深灰（#1A2332）背景，契合通信网络运维场景
- **辅色调**：绿色（#00E676）表示健康/正常、橙色（#FF9100）表示预警/注意、红色（#FF1744）表示严重/故障
- **按钮风格**：圆角8px、微渐变、hover时发光效果
- **字体**：标题使用 DIN Alternate（数字仪表盘风格），正文使用 Noto Sans SC
- **布局风格**：左侧固定导航栏 + 顶部筛选栏 + 主内容区域，卡片式模块布局
- **图标风格**：线性图标（lucide-react），统一2px描边
- **动效**：数据刷新时渐变过渡、卡片hover轻微浮起、预警闪烁动画、热力图渐变着色

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|

### 4.3 响应式设计

- 桌面端优先（1440px+），适配1280px最小宽度
- 侧边栏可折叠，小屏时自动收起为图标模式
- 卡片网格在小屏时从多列变为单列
- 图表在小屏时保持最小宽度并允许横向滚动

### 4.4 3D场景指导

不适用，本项目以2D数据可视化为主。
